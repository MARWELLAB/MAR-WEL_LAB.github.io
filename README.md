## 📁 File Structure
```
MAR-WEL_LAB.github.io/
│
├── index.html          ← Home page
├── about.html          ← About the lab
├── research.html       ← Research areas
├── projects.html       ← Funded projects
├── publications.html   ← Journal & conference papers
├── team.html           ← Faculty, scholars, students
├── contact.html        ← Contact form + map
│
├── css/
│   └── style.css       ← All styles (responsive, mobile-ready)
│
├── js/
│   └── script.js       ← Navigation, animations, form handling
│
├── images/
│   ├── marwel_logo.png ← Lab logo
│   ├── banner.jpg      ← Hero banner (add your own)
│   └── team/           ← Team member photos (add your own)
│
└── README.md
```

## 🚀 How to Deploy on GitHub Pages

1. Create a GitHub account / org named `MAR-WEL_LAB`
2. Create a new repository named `MAR-WEL_LAB.github.io`
3. Upload all these files to the repository
4. Go to **Settings → Pages → Source: main branch / root**
5. Your site will be live at `https://MAR-WEL_LAB.github.io`

## ✏️ How to Customize

| What to update | Where |
|----------------|-------|
| PI name & photo | `team.html` + `images/team/` |
| Real team names | `team.html` |
| Real publications | `publications.html` |
| Project funding details | `projects.html` |
| Lab phone/email | `contact.html` |
| Stats numbers | `index.html` (data-target values) |
| Logo | Replace `images/marwel_logo.png` |

- No frameworks or build tools needed
- Fully responsive (mobile, tablet, desktop)
